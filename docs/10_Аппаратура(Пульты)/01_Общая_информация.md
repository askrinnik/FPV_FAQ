# Аппаратура/пульты

## Первичная настройка аппаратуры на Edge TX 
От @Konstantin_Gr  
1. Выбираешь модель: фпв дрон.  
2. Включаешь внутренний приёмник  
3. Покетрейт на 250 Гц без букв спереди  
4. ADC фильтр отключить
